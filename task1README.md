# Task 1: Sales Data Analysis

## Overview
This task involves analyzing a supermarket sales dataset to extract valuable insights for business decision-making. The goal is to perform data cleaning, exploratory data analysis (EDA), and visualize key trends in sales.

## Steps:
1. **Load the Dataset**: The dataset is loaded from a CSV file (`supermarket_sales.csv`).
2. **Data Cleaning**: Handle missing values by filling them with the mean of the respective columns and remove duplicates.
3. **Exploratory Data Analysis (EDA)**: 
   - Analyze the total sales by product category.
   - Determine the city with the most customers.
   - Calculate the average purchase value.
4. **Visualizations**: 
   - Bar plot showing sales by product category.
   - Count plot showing the number of customers by city.
   - Histogram for the distribution of purchase values.

## Requirements:
- pandas
- matplotlib
- seaborn
